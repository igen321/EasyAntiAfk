How to use the EASYAFK script (requires AutoHotkey v1):
Download and install AutoHotkey v1 from https://www.autohotkey.com/


1. Open the script by clicking the EasyAntiAfk 0.0.x.ahk file.
2. Set the maximum wait time (in seconds) in the "Max wait" field.
3. Set the key hold duration (in milliseconds) in the "Key hold" field.
4. Click "Start" to activate EasyAntiAfk
5. The script will randomly press one of the keys: W, A, S, D, or Space.
6. Logs of key presses will appear in the text box for reference.
7. Click "Stop" to pause the script at any time.
8. Press Esc or close the window to exit the script.
